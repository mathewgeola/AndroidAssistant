# AndroidAssistant

安卓助手

### 创建项目

~~~bash
python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple

pyside6-project new-ui AndroidAssistant

cd AndroidAssistant

pyside6-project build

pyside6-project run
~~~

### 编译项目

~~~bash
# win
.\build.bat

# mac or linux
./build.sh
~~~
